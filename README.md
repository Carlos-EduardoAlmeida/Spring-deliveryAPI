# Spring - Delivery API
API made in Java/Spring to be used in a delivery service, with registration, editing and possibility of deleting: user, address and order. there is also an address query by zip code using an external API

# Techonologies
• Java<br>
• Spring boot<br>
• PostgreSQL<br>
• Swagger, JPA, Flyway
